## Usage

To use the `UserProfile` component in your project, import it as follows:

```tsx
import UserProfile from '@gaguirr5/mcxlib/components/userprofile';

const Example = () => {
  return <UserProfile username='Jane Doe' userEmail='Jane@doe.com' />;
};

export default Example;
```
